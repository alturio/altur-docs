## Bienvenido a la API de Altur

Esta guía contiene información detallada y actualizada de los endpoints disponibles para gestionar las comunicaciones de tu organización.

### Alcance Actual

Hoy la API de Altur está enfocada principalmente en la integración de mensajería. Aún no cubre toda la funcionalidad para administrar chatbots, campañas o ajustes de la organización. Para esas tareas usa la app web de Altur.

<Card title="Altur App" icon="browser" href="https://app.altur.io">
  Abre la app de Altur en la web
</Card>

### Por qué usar la API de Altur

Con la API de Altur puedes:
- **Automatizar Conversaciones**: integrar mensajería para flujos de usuario sin fricción.
- **Recibir Datos en Tiempo Real**: usar webhooks para enterarte de eventos importantes, como cambios de estado de mensajes o el fin de una llamada.
- **Personalizar Flujos**: combinar endpoints para construir integraciones a la medida de tu negocio.

Para la lista completa de endpoints y ejemplos, navega las secciones de abajo.
