Las campañas son una funcionalidad clave de Altur que permite iniciar conversaciones a gran escala de manera automática, sin sacrificar la personalización en cada interacción.

## Tipo de Campaña

Actualmente, Altur ofrece dos canales para campañas: **Llamadas** y **WhatsApp**.

<Warning>
  Recomendamos utilizar campañas de **WhatsApp** únicamente cuando tu base de contactos haya dado su consentimiento previo para recibir mensajes de tu empresa. Esto se debe a las estrictas regulaciones sobre el envío de mensajes outbound masivos, cuyo incumplimiento puede derivar en sanciones o bloqueos de tu cuenta.
</Warning>

## Configuración de una campaña

Sigue estos pasos para crear una nueva campaña en Altur:

1. **Accede a la vista de campañas:** Dirígete a la sección de **Campañas** que encontrarás en la barra lateral izquierda y haz click en **Nueva Campaña** en la esquina superior derecha.

<Note>
  Si no tienes alguna de estas opciones, solicita con el administrador de tu organización los permisos necesarios.
</Note>

2. **Completa los campos básicos:**
   - **Nombre de la campaña:** Escribe un nombre corto y descriptivo.
   - **Descripción:** Agrega detalles sobre el objetivo de la campaña (para uso interno).

3. **Selecciona un agente:** Elige el **agente** configurado previamente para esta campaña. Este **agente** será el encargado de gestionar las conversaciones.

4. **Selecciona una integración:** Selecciona la línea telefónica o número de WhatsApp desde donde se enviarán los mensajes o llamadas.

<Note>
  Si vas a usar WhatsApp, asegúrate de que la cuenta y el número estén configurados e integrados a Altur correctamente.
</Note>


5. **Define la zona horaria** Selecciona la zona horaria correspondiente a tus contactos. Por defecto, se selecciona **America/Mexico_City**.

6. **Mensaje inicial (Llamadas):** Escribe el mensaje inicial, esta frase será lo primero que dirá el Agente al conectar la llamada.

7. **Plantilla (WhatsApp):** Selecciona la plantilla (WhatsApp), elige una o dos plantillas de mensaje aprobadas. Si la plantilla seleccionada contiene un encabezado con imagen, se habilitará el campo para subirla. Asegúrate de que coincida con el contenido aprobado por Meta.

8. **Configura los reintentos:** Define cuántas veces se debe **reintentar** enviar un mensaje o realizar una llamada en caso de que el contacto no responda o se haya alcanzado buzón de voz.

9. **(Opcional) Configura las vueltas (Llamadas):** Opcionalmente, habilita vueltas en la campaña. Permite que la campaña se repita automáticamente si quedan contactos por convertir. Si activas esta opción, podrás configurar:
    - Número máximo de vueltas al día
    - Tiempo de espera entre vueltas (en minutos)
    - Filtros de estatus (e.g. evitar volver a contactar a personas que ya contestaron)
    - Filtros de etiquetas (como contactos marcados como número incorrecto)

10. **Configura límite de envíos diarios (WhatsApp):** Indica cuántos mensajes pueden enviarse al día. Esto evita saturar la integración o exceder límites impuestos por WhatsApp.

11. **Guarda la campaña:** Haz click en **Guardar** para crear la nueva campaña con la configuración establecida.

<Check>
  ¡Listo\! Has creado una nueva campaña.
</Check>

## Preparación del Archivo CSV

Tu archivo CSV debe incluir las siguientes **columnas**:

| Columna   | Tipo   | Requerido | Descripción                                      |
| --------- | ------ | --------- | ------------------------------------------------ |
| `name`    | string | Sí        | Nombre del contacto (usado para personalización) |
| `contact` | string | Sí        | Número en formato E.164                          |
| `context` | string | Sí        | Texto libre o JSON con contexto para el agente   |
| `id`      | string | No        | ID interno para referencia cruzada               |

## Iniciar tu Campaña

Una vez que tu campaña esté configurada y hayas subido el CSV:

1. **Revisa** la pantalla de resumen para confirmar la configuración y cantidad de contactos.
2. **Haz clic en "Iniciar Campaña"** para comenzar con las llamadas o envíos según el horario.
3. **Monitorea el Progreso** en tiempo real desde el Panel de Campaña:
   - Visualiza número de interacciones.
   - Rastrea fallos, buzones de voz y vueltas de reintento.
4. **Descarga los Resultados** al finalizar:
   - Exporta un CSV completo con datos extraídos, resultados de gestión y estados.
   - Aplica filtros (ej. convertidos, no contactados) para afinar tus listas de seguimiento.
5. **Repetir o Archivar**:
   - Si tu campaña usa vueltas, los ciclos se ejecutarán automáticamente.
   - De lo contrario, puedes volver a subir un CSV filtrado o archivar la campaña al terminar.
