## Authentication

To use our API, you need to authenticate your requests by adding an `Authorization` header with your secret key. Secret keys can be generated from within the [Altur app dashboard](https://app.altur.io/dashboard/api).

### Example Authentication Header

```curl
Authorization: Bearer YOUR_API_SECRET_KEY
```
